# \[Rogue]

<figure><img src="../../.gitbook/assets/700px-1Arruaceiro.png" alt=""><figcaption></figcaption></figure>

* เช่นเดียวกับคนขี้โกง พวกเขาสามารถถอดเกราะของคู่ต่อสู้ได้ในพริบตาขณะยังทำความเสียหายได้มาก.
