# Various Issues

## **Clicking Download Does Not Download Arkaik**

**Problem:**

* Whether using the installer or the ZIP file, **Arkaik does not download at all**.

**Solution:**

* Search for **"Tablet PC Settings"**
* In the first option, set it to **"Never use tablet mode"**
